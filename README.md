### CSS
#### 引入方式: 1.外部引入(link标签)。2.内部样式(style标签)。3.行间样式(style属性)。
#### 引入方式优先级: 行间样式 > 内部样式 > 外部引入
#### 选择器优先级: id选择器 > 类选择器 > 标签选择器 > 子选择器 > 后代选择器 > 伪类选择, 需要注意的是: !important的优先级是最高的。
